# Customer Churn Prediction & Lifetime Value (LTV) Engine

## Project Overview

This project focuses on predicting customer churn and estimating Customer Lifetime Value (LTV) using machine learning. The goal is to help businesses identify customers who are likely to leave, understand the key factors behind churn, and support better customer retention through data-driven insights.
